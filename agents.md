## üìù Agent Configuration

### Available Agents

#### general (General-Purpose Agent)
- **Purpose:** Research, multi-step tasks, parallel execution
- **Use for:** Complex research, exploring codebase, executing multiple units
- **Best when:** Task is not well-defined or requires exploration

#### explore (Exploration Agent)
- **Purpose:** Fast codebase exploration and pattern finding
- **Use for:** Finding files by patterns, searching code, understanding structure
- **Thoroughness levels:** quick, medium, very thorough
- **Best when:** You need to find specific files or understand code organization

---

### When to Use Which Agent

| Task Type | Recommended Agent | Why |
|-----------|------------------|-----|
| **Find all React components** | explore (medium) | Fast pattern matching for file finding |
| **Research React Native best practices** | general | Multi-step research task |
| **Analyze specific component** | Read tool directly | Single file, no need for agent |
| **Search for bug pattern** | grep tool | Direct content search |
| **Generate multiple related files** | general | Parallel execution benefit |
| **Review entire codebase** | explore (very thorough) | Comprehensive analysis |
| **Create PRD from plan** | No agent needed | Single file creation |

## ü§ñ AI Task Generation Tips

If you use AI to generate tasks from a PRD:

1. **Define task template first:**
   ```markdown
   - [ ] ID: Task name [PRIORITY] [TIME_EST] ‚Üí Depends on: ID
     - Description: What needs to be done
     - Acceptance criteria: [ ] [ ] [ ]
     - Files: path/to/file.tsx
   ```

2. **Create separate tracks:**
   - Development tasks
   - Testing tasks
   - Documentation tasks
   - Marketing/Business tasks

3. **Add realistic dependencies:**
   - Setup ‚Üí Build ‚Üí Test ‚Üí Deploy
   - Database ‚Üí API ‚Üí Frontend
   - Components ‚Üí Screens ‚Üí Navigation

4. **Include non-feature work:**
   - Configuration
   - Infrastructure
   - DevOps
   - Asset creation

5. **Review and validate:**
   - Are tasks actionable?
   - Are priorities correct?
   - Are dependencies logical?
   - Is the effort estimate realistic?
